# Exploratory Data Analysis using SQL

This project explores trends and insights from a tech layoffs dataset using SQL.

## Analysis Performed
- Company-wise and country-wise layoffs
- Yearly and monthly layoff trends
- Rolling totals using window functions
- Top companies by layoffs per year
- Stage-wise and funding-based insights

## Tools Used
- MySQL
- SQL (aggregations, CTEs, window functions)

## Outcome
Identified key patterns and trends to support data-driven decision making.
